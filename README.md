# Calculator
🧮 Simple Python Calculator

This is a beginner-friendly calculator program written in Python. It takes two numbers as input from the user and performs basic arithmetic operations like addition, subtraction, multiplication, and division.

🚀 Features
Takes user input for two numbers
Supports operations: +, -, *, /
Simple and easy-to-understand logic
Perfect for Python beginners
💻 How It Works
User enters first number
User enters second number
User selects an operation
Program displays the result
